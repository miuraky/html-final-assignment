# UrbanKicks - スニーカー管理・情報サイト

## 概要
スニーカーヘッズのためのコレクション管理、情報収集、ケアガイド、コミュニティ機能を提供するWebサイトです。
「運命の一足を、もっと身近に。」をコンセプトに、スニーカーライフをサポートします。

## ページ構成
1. **Home (index.html)**: サイトのトップページ。最新のピックアップやサービスの特徴を紹介。
2. **Collection (sneaker-list.html)**: スニーカー図鑑。ブランドやレア度で検索・絞り込みが可能。
3. **Care Guide (care-guide.html)**: スニーカ－のお手入れ方法を解説。
4. **FAQ (sneaker-faq.html)**: よくある質問と回答。ユーザー間のQ&A機能（想定）。
5. **Style Log (coordinate-log.html)**: 日々の着用記録やコーディネートを管理。
6. **Wish List (wish-list.html)**: 欲しいスニーカーのリスト管理と発売情報。
7. **Contact (contact.html)**: お問い合わせフォーム。
8. **Login (login.html)**: ユーザーログインページ（新規追加）。
9. **Sign Up (register.html)**: 新規会員登録ページ（新規追加）。
10. **Detail (sneaker-detail.html)**: スニーカー詳細ページ。
11. **About (about.html)**: サイトについて。

## Bootstrap適用箇所と内容
レイアウト変更とUI構築のためにBootstrap 5.3を活用しました。

### 1. カルーセル（スライドショー）の導入
*   **適用箇所**: `index.html` (Home) の「今月のピックアップ」セクション。
*   **内容**: `carousel`, `carousel-item`, `slide` クラスを使用し、注目のスニーカー3足を自動で切り替えて表示。動きのあるトップページを演出しました。

### 2. レスポンシブグリッドシステムの調整
*   **適用箇所**: `sneaker-list.html` (Collection) の一覧表示。
*   **内容**: `row` クラスの中に `col-lg-3` (PCで4列), `col-md-6` (タブレットで2列)、`col-12` (スマホで1列) を適用。
*   **効果**: 画面サイズに合わせてレイアウトが自動で切り替わるレスポンシブ対応を行い、多くのアイテムを効率よく見られるようにしました。

### 3. カードコンポーネントとバッジ
*   **適用箇所**: 全ページのスニーカー表示部、ウィッシュリストなど。
*   **内容**: `card`, `card-body`, `badge` クラスを使用。画像、タイトル、説明文、タグ（ブランド名など）を美しく整理して表示しました。

### 4. アコーディオンメニュー
*   **適用箇所**: `care-guide.html`, `sneaker-faq.html`。
*   **内容**: `accordion` クラスを使用し、クリックすると詳細が開閉するUIを実装。情報量が多いページでもすっきりと見せることができます。

### 5. フォームとバリデーション
*   **適用箇所**: `contact.html`, `login.html`, `register.html`。
*   **内容**: `form-control`, `input-group` を使用して整った入力フォームを作成しました。

### 6. ナビゲーションバー (Navbar)
*   **適用箇所**: 全ページ上部。
*   **内容**: `navbar-expand-lg` を使用し、PCでは横並び、スマホではハンバーガーメニューに自動で切り替わるレスポンシブなメニューを実装しました。

## 技術的なポイント (components.js)
保守性（メンテナンスのしやすさ）を高めるために、`js/components.js` を作成しました。

*   **役割**: サイト内の全ページで共通して使われる「ヘッダー（ナビゲーション）」と「フッター」を一元管理しています。
*   **仕組み**: HTML側には `<div id="navbar-placeholder"></div>` という空のコンテナを用意し、JavaScriptがその中にヘッダーのHTMLコードを自動で挿入します。
*   **メリット**: 「メニューのリンクを1つ変えたい」といった場合に、全HTMLファイルを修正する必要がなく、`components.js` を1箇所修正するだけで全ページに反映されます。

## 変更箇所まとめ
*   **テーマの刷新**: ペット管理サイトからスニーカー管理サイトへ、画像・テキスト・配色をフルリニューアル。
*   **新規ページ**: `login.html`, `register.html` を追加し、ログインフローを意識した導線を作成。
*   **共通化**: ヘッダーとフッターを `js/components.js` に切り出し、管理コストを削減。
*   **日本語化**: 全ページのテキストを英語から自然な日本語へ翻訳・修正。

## 工夫した点・苦労した点
*   **工夫した点**:
    *   **世界観の統一**: カラーコード（ブラウン、オレンジ、ダークグレー）をCSS変数で定義し、Bootstrapのユーティリティクラスと組み合わせることで、「UrbanKicks」独自の世界観をBootstrapベースで表現しました。
    *   **AIを活用した効率的な実装**: ページ数が多いため、ヘッダーやフッターをJavaScriptで共通化（コンポーネント化）しました。この際、AIツールを活用してコードの構造化や整理を行うことで、短時間で保守性の高いコードを実装しました。
*   **苦労した点**:
    *   **グリッドの調整**: スマホ、タブレット、PCそれぞれの画面サイズで崩れないように、ブレークポイント（`md`, `lg`）ごとのカラム数を調整するのに試行錯誤しました。
