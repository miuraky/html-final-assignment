# KicksLog - スニーカーコレクション管理サイト

## 概要
スニーカーコレクターのための管理・情報共有サイトです。
所持しているスニーカーの管理、コーディネートの記録、メンテナンス方法の確認などができます。

## オリジナル要素と工夫した点

### 1. サイト全体のテーマ変更
- **Before**: ペット管理サイト（犬・猫・小動物）
- **After**: スニーカー管理サイト（コレクション・コーデ・ケア）
- 配色をアースカラー（茶色・緑）から、ストリート感を意識したモノトーンベース＋アクセントカラーに変更しました。
- 画像プレースホルダーやアイコンをすべてスニーカー関連のものに差し替えました。

### 2. コンポーネント化によるリファクタリング
- ナビゲーションバーとフッターを `js/components.js` に共通化。
- 各HTMLファイルから重複コードを削除し、メンテナンス性を向上させました。
- ページごとのActive状態の切り替えもJavaScriptで自動化しています。

### 3. Bootstrapの活用とカスタマイズ
- **グリッドシステム**: コレクション一覧やガイドページでレスポンシブな配置を実現。
- **カードコンポーネント**: スニーカー詳細や記事一覧を見やすく整理。
- **アコーディオン**: FAQやケアガイドで情報をコンパクトに収納。
- **バッジ・ボタン**: Bootstrapのユーティリティクラスを活用し、統一感のあるデザインに。
- CSS変数 (`:root`) を導入し、テーマカラーの一括管理を行っています。

### 4. 機能的なページ構成
- **index.html**: ヒーローヘッダー、注目のスニーカー、ブランド一覧。
- **about.html**: サイトのミッションと概要（新規作成）。
- **sneaker-list.html**: フィルタリング機能付きのコレクション一覧。
- **care-guide.html**: アコーディオンを活用したメンテナンス手順書。
- **coordinate-log.html**: 着用履歴と統計データの可視化。
- **sneaker-faq.html**: タブ切り替えによるカテゴリ別Q&A。
- **contact.html**: フォームバリデーション付きのお問い合わせ。
- **sneaker-detail.html**: 詳細スペックとマルチアングル画像（新規追加）。
- **wish-list.html**: 優先度付きの欲しいものリスト（新規追加）。

## 変更履歴 (2026-01-25)
